# Kanban Board Project

A responsive Kanban-style project management app built with **React**, **Tailwind CSS** and **dnd-kit** for drag-and-drop functionality. The project emphasizes modular components, responsive design, and clean state management.

---

## Live Preview

Check out the live application here: [Kanban Board Demo](https://kanban-board-bice-xi.vercel.app/)

---

## Features

- **Boards, Columns, and Tasks**: Create, edit, delete, and manage boards, columns, and tasks.
- **Drag & Drop**: Move tasks between columns with smooth animations using `@dnd-kit`.
- **Responsive UI**: Layout adjusts for small, medium, and large devices.
- **Task Management**: Editable titles and descriptions with auto-resizing textareas.
- **Delete/Modify Actions**: Contextual delete buttons with proper responsive handling.
- **Local Storage**: Boards and selected board indices persist between sessions.
- **Optional Enhancements**: Dark mode, accessibility improvements, and unit testing.

---

## Technologies Used

- **React** (Vite)
- **TailwindCSS**
- **dnd-kit**: Drag-and-drop library
- **Radix UI**: Headless components
- **Immer**: Immutable state management
- **Context API**: Global state management
- **Optional**: TypeScript (future migration)
